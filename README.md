# Nobara Automation Scripts

![Test](https://github.com/raizenica/noba/actions/workflows/test.yml/badge.svg)

A collection of automation scripts for my Nobara Linux (Fedora) system.  
They are designed to be used both from the command line and integrated into Dolphin file manager as service menus.

## 🔧 Installation

Clone the repository and make the scripts executable:

```bash
git clone https://github.com/raizenica/noba.git ~/bin/noba
cd ~/bin/noba
chmod +x *.sh
